vk-photo-downloader
===

a slow python script for downloading photos from vk.com

### Usage
1) Get the photo links ,and save in text.
  - e.g. https://vk.com/photo-XXXXXXXX_YYYYYYYYY

2) Set the output path.

3) Run, wait, and done.
